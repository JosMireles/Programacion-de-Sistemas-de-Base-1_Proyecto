Projecto de Texto a Voz (TTS, Text-To-Speech)

Descripcion:

Este proyecto implementa una gramática en ANTLR orientada al reconocimiento y análisis sintáctico de oraciones en español. La gramática distingue entre tres tipos principales de enunciados: preguntas, exclamaciones y declarativas, permitiendo tanto la forma normativa (con signos de apertura ¿, ¡) como la forma flexible usada comúnmente en textos informales.

El repositorio incluye:
- Definición completa de reglas de parser y lexer en EaV2.g4.
- Soporte para palabras con acentos y caracteres propios del español (á, é, í, ó, ú, ñ).
- Reconocimiento de números y conjunciones (y, o, pero).
- Ejemplos de entrada para probar la gramática con grun y visualizar árboles de análisis.

Este proyecto sirve como base para explorar el procesamiento de lenguaje natural en español, mostrando cómo ANTLR puede adaptarse tanto a las normas lingüísticas como al uso cotidiano.
